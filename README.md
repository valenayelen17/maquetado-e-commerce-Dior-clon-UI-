# Dior E-commerce Clone UI

Proyecto de maquetado web inspirado en el sitio oficial de Dior.

## Tecnologías utilizadas
- Diseño Responsive
- Maquetado Mobile First

## Características
- Home con banners dinámicos
- Sección de productos
- Página de detalle de producto
- Diseño adaptable a Desktop, Tablet y Mobile
- Footer estructurado

## Objetivo
Practicar maquetado profesional replicando la interfaz de un e-commerce real, trabajando estructura semántica y diseño responsive.
